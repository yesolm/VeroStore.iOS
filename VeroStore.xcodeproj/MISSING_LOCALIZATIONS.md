# Missing Localization Strings

Add these to your Localizable.strings files:

## English (en.lproj/Localizable.strings)

```
// Search
"recent_searches" = "Recent Searches";
"clear_all" = "Clear All";

// Product Detail
"add_to_cart" = "Add to Cart";
"added_to_cart" = "added to cart";
"select_options" = "Select Options";
"please_select_option" = "Please select an option";
"quantity" = "Quantity";

// Common
"in_stock" = "In Stock";
"out_of_stock" = "Out of Stock";
```

## Amharic (am.lproj/Localizable.strings)

```
// Search
"recent_searches" = "የቅርብ ጊዜ ፍለጋዎች";
"clear_all" = "ሁሉንም አጽዳ";

// Product Detail
"add_to_cart" = "ወደ ጋሪ ጨምር";
"added_to_cart" = "ወደ ጋሪ ታክሏል";
"select_options" = "አማራጮችን ምረጥ";
"please_select_option" = "እባክዎ አማራጭ ይምረጡ";
"quantity" = "ብዛት";

// Common
"in_stock" = "በመጋዘን ውስጥ";
"out_of_stock" = "ከመጋዘን ወጥቷል";
```

## How to Add:

1. In Xcode, find your Localization folder
2. Edit `en.lproj/Localizable.strings`
3. Add the English strings above
4. Edit `am.lproj/Localizable.strings`
5. Add the Amharic strings above
6. Build the project
